## [1.0.0] - 2025-01-12

### Added

- Initial release of Safari MCP server
- **Tab Management**: List all open tabs, get active tab, open new tabs, close tabs, switch between tabs, reload tabs
- **Window Management**: List all windows, create new windows, close windows
- **Bookmark Operations**: List bookmarks, add bookmarks, remove bookmarks, search bookmarks
- **History Access**: Search browsing history, get recent history, clear history
- **Reading List**: Add URLs to reading list, list reading list items, remove items
- **Page Interaction**: Get page content, execute JavaScript on pages
- Model Context Protocol (MCP) integration for AI assistant compatibility
- macOS AppleScript integration for Safari automation
- TypeScript implementation with full type safety
- Comprehensive test suite with Vitest
